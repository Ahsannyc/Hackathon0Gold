
## Getting Credentials Securely

### Google OAuth (Gmail)
1. Go to [Google Cloud Console](https://console.cloud.google.com/)
2. Create OAuth 2.0 credentials (Desktop Application)
3. Download credentials as JSON
4. Place in project root as `credentials.json`
5. Run `python authenticate_gmail.py` to generate token

### Twilio (WhatsApp)
1. Go to [Twilio Console](https://console.twilio.com/)
2. Get your Account SID and Auth Token
3. Add to `.env`:
   ```
   TWILIO_ACCOUNT_SID=your_sid
   TWILIO_AUTH_TOKEN=your_token
   ```

### Twitter/X API
1. Apply at [Twitter Developer Portal](https://developer.twitter.com/)
2. Add to `.env`:
   ```
   TWITTER_API_KEY=your_key
   TWITTER_API_SECRET=your_secret
   TWITTER_BEARER_TOKEN=your_token
   ```

## What Happens If Secrets Are Accidentally Committed?

If you ever see real secrets in git:

```bash
# 1. Immediately rotate the credential in the service (Google Cloud, Twilio, etc)
# 2. Remove from git history using git-filter-branch:
git filter-branch --force --index-filter \
  'git rm --cached --ignore-unmatch credentials.json' \
  --prune-empty --tag-name-filter cat -- --all

# 3. Force push to remote:
git push origin --force --all
git push origin --force --tags
```
